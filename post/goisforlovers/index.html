使用Hugo和GitHub pages搭建网页（BLOG、NEWS）

@徐其琛 · Jan 10, 2018 · 1 min read

Hugo是一种用go语言写成静态网站框架，使用Hugo可以实现个人网站、会议网站、公司网站以及团队网站等各类网站的快速生成，搭配使用GitHub pages则可以优雅地部属生成的网站。

Hugo在windows系统下的下载，安装及测试

Hugo官网推荐使用Chocolatey包管理工具来进行快速安装，具体代码如下：

    choco install hugo -confirm

但如果使用的机器上并未安装有Chocolatey包管理工具，那可以通过以下方案安装：

1、从Hugo官方网站中的Hugo Releases处下载最新版本（推荐使用64位版本即-64bit版本）,其地址为：

https://github.com/gohugoio/hugo/releases

2、解压该压缩文件，并将其添加至windows系统环境变量中，具体流程如下：

添加环境变量：

注意这里的C:\hugo\bin为你的解压目录

3、测试是否安装成功：

在powershell输入代码 hugo version

如果出现以上字样，则代表已完成hugo的安装。

制作网站

1、新建一个文件夹，用于保存下载的主题、你撰写的文字以及网站所需的图片。

2、在powershell中输入：

    hugo new site xxxxx

xxxxx为该网站的名称，诸君随意

如果一切顺利，该文件夹下会出现一个xxxxx为名的新文件夹，进入该文件夹，你会发现Hugo以及帮你建立了如下目录：

在该目录下打开powershell 输入git命令克隆所需要的主题：

主题可从https://themes.gohugo.io/中获得

git clone https://github.com/----------------------

如图我们下载一个名叫Academic的主题

注意，上图中有一个错误，一般不用中文作为路径名，这可能会导致一些莫名错误

3、根据主题的介绍更改内容设置网页，需注意尽管config.toml文件是重点，但每个主题的编辑方式不一，务必认真阅读各主题的文档。

4、在网站根目录打开powershell 输入hugo server ，即可在 http://localhost:1313 即时欣赏写好的网站。

部署写好的静态网页到GitHub pages

1、在你的GitHub中建立专属Repository，命名为：你的用户名.github.io

2、在你建立的网站的根目录打开powershell 输入：

     hugo --theme=xxxxx --buildDrafts --baseUrl="https://yyyyy.github.io/"

其中xxxxx为你选用的主题，yyyyy为你的github用户名 3、输入以下命令

cd public
git init
git remote add origin git@你的ssh
git add -A
git commit -m "first commit"
git push -u origin master

4、连上外网在访问https://yyyyy.github.io/

5、Enjoy！
